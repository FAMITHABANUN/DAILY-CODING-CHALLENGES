# Day 14 - LeetCode Solutions

## Problem 1

- Problem Name: Department Top Three Salaries
- Platform: LeetCode
- Language: SQL
- Difficulty: Hard
- Approach: Use DENSE_RANK() with PARTITION BY to rank employee salaries within each department and retrieve the top three salaries.
- Time Complexity: O(n log n)
- Space Complexity: O(1)

## Problem 2

- Problem Name: Intersection of Two Arrays
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Use HashSet to store elements and find the unique intersection between two arrays efficiently.
- Time Complexity: O(n + m)
- Space Complexity: O(n)

## Problem 3

- Problem Name: Third Maximum Number
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Track the first, second, and third distinct maximum values while traversing the array.
- Time Complexity: O(n)
