# Day 08 - LeetCode Solutions

## Problem 1

- Problem Name: Median of Two Sorted Arrays
- Platform: LeetCode
- Language: Java
- Difficulty: Hard
- Approach: Use binary search to efficiently find the median of two sorted arrays.
- Time Complexity: O(log(min(m, n)))
- Space Complexity: O(1)

## Problem 2

- Problem Name: Merge Two Sorted Lists
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Compare the elements of both linked lists and merge them into a single sorted list.
- Time Complexity: O(n + m)
- Space Complexity: O(1)

## Problem 3

- Problem Name: Employee Bonus
- Platform: LeetCode
- Language: SQL
- Difficulty: Easy
- Approach: Use LEFT JOIN to combine Employee and Bonus tables and filter the required results.
- Time Complexity: O(n)
- Space Complexity: O(1)
