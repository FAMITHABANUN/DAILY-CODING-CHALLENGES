# Day 15 - LeetCode Solutions

## Problem 1

- Problem Name: Convert a Number to Hexadecimal
- Platform: LeetCode
- Language: Java
- Difficulty: Easy

### Approach
- Use bitwise operations to convert the given decimal number into its hexadecimal representation.
- Handle both positive and negative integers according to the problem constraints.

### Time Complexity
- O(log n)

### Space Complexity
- O(1)

- Platform: LeetCode

## Problem 3

- Problem Name: Department Highest Salary
- Platform: LeetCode (SQL)
- Language: SQL
- Difficulty: Medium

### Approach
- Use `JOIN` operations to combine the Employee and Department tables.
- Find the maximum salary for each department using `GROUP BY` and `MAX()`.
- Retrieve employees whose salary matches the highest salary in their respective department.

### Time Complexity
- O(n)

### Space Complexity
- O(1)
