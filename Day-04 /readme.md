# Day 04 - LeetCode Solutions

## Problem 1

**Problem Name:** Remove Element

**Platform:** LeetCode

**Difficulty:** Easy

**Approach:** Use two pointers to overwrite the elements that are not equal to the given value and return the new length.

**Time Complexity:** O(n)

**Space Complexity:** O(1)

---

## Problem 2

**Problem Name:** Find the Index of the First Occurrence in a String

**Platform:** LeetCode

**Difficulty:** Easy

**Approach:** Compare the substring starting from each index of the main string until the target string is found.

**Time Complexity:** O(n × m)

**Space Complexity:** O(1)

---

## Problem 3

**Problem Name:** Rank Scores

**Platform:** LeetCode (SQL)

**Difficulty:** Medium

**Approach:** Use the `DENSE_RANK()` window function to assign rankings without gaps for duplicate scores.

**Time Complexity:** O(n log n)

**Space Complexity:** O(1)
