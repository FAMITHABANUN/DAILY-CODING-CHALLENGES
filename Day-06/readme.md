# Day 06 - LeetCode Solutions

## Problem 1

**Problem Name:** 4Sum

**Platform:** LeetCode

**Language:** Java

**Difficulty:** Medium

**Approach:** Sort the array and use nested loops along with the Two Pointer technique to find all unique quadruplets.

**Time Complexity:** O(n³)

**Space Complexity:** O(1)

## Problem 2

**Problem Name:** Duplicate Emails

**Platform:** LeetCode (SQL)

**Language:** SQL

**Difficulty:** Easy

**Approach:** Use GROUP BY and HAVING COUNT(*) > 1 to identify duplicate email addresses.

**Time Complexity:** O(n)

**Space Complexity:** O(1)
