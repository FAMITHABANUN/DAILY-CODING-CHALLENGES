# Day 11 - LeetCode Solutions

## Problem 1

- Problem Name: Customers Who Never Order
- Platform: LeetCode
- Language: SQL
- Difficulty: Easy
- Approach: Use a subquery with NOT IN to find customers whose IDs are not present in the Orders table.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

- Problem Name: Remove Nth Node From End of List
- Platform: LeetCode
- Language: Java
- Difficulty: Medium
- Approach: Use the Two Pointer technique to locate and remove the nth node from the end in a single traversal.
- Time Complexity: O(n)
- Space Complexity: O(1)
