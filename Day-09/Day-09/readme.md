# Day 09 - LeetCode Solutions

## Problem 1

- Problem Name: Length of Last Word
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Traverse the string from the end, skip trailing spaces, and count the characters of the last word.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

- Problem Name: Multiply Strings
- Platform: LeetCode
- Language: Java
- Difficulty: Medium
- Approach: Simulate the multiplication process digit by digit without converting the strings to integers.
- Time Complexity: O(m × n)
- Space Complexity: O(m + n)

## Problem 3

- Problem Name: Swap Salary
- Platform: LeetCode
- Language: SQL
- Difficulty: Easy
- Approach: Use the CASE statement to swap the values of 'm' and 'f' in the sex column.
- Time Complexity: O(n)
- Space Complexity: O(1)
