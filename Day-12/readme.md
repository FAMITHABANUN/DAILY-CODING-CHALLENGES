# Day 12 - LeetCode Solutions

## Problem 1

- Problem Name: Find Numbers with Even Number of Digits
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Count the number of digits in each integer and check whether it is even.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

- Problem Name: Running Sum of 1D Array
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Traverse the array once and keep adding the previous sum to the current element.
- Time Complexity: O(n)
- Space Complexity: O(1)
