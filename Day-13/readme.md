# Day 13 - LeetCode Solutions

## Problem 1

- Problem Name: Employees Earning More Than Their Managers
- Platform: LeetCode
- Language: SQL
- Difficulty: Easy
- Approach: Use a SELF JOIN on the Employee table to compare an employee's salary with their manager's salary.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 2

- Problem Name: Missing Number
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Calculate the expected sum of numbers from 0 to n and subtract the actual array sum to find the missing number.
- Time Complexity: O(n)
- Space Complexity: O(1)

## Problem 3

- Problem Name: Move Zeroes
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Use the two-pointer technique to move all zeroes to the end while maintaining the relative order of non-zero elements.
- Time Complexity: O(n)
- Space Complexity: O(1)
