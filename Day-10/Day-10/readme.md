# Day 10 - LeetCode Solutions

## Problem 1

- Problem Name: Pow(x, n)
- Platform: LeetCode
- Language: Java
- Difficulty: Medium
- Approach: Use Binary Exponentiation (Fast Power) to calculate xⁿ efficiently.
- Time Complexity: O(log n)
- Space Complexity: O(log n)

## Problem 2

- Problem Name: Sqrt(x)
- Platform: LeetCode
- Language: Java
- Difficulty: Easy
- Approach: Use Binary Search to find the integer square root of the given number.
- Time Complexity: O(log n)
- Space Complexity: O(1)

## Problem 3

- Problem Name: User Activity for the Past 30 Days I
- Platform: LeetCode
- Language: SQL
- Difficulty: Easy
- Approach: Use GROUP BY with COUNT(DISTINCT user_id) to calculate the number of active users for each day in the last 30 days.
- Time Complexity: O(n)
- Space Complexity: O(1)
