# Day 07 - LeetCode Solutions

## Problem 1

**Problem Name:** Delete Duplicate Emails

**Platform:** LeetCode (SQL)

**Language:** SQL

**Difficulty:** Easy

**Approach:** Use SQL DELETE statements along with subqueries to remove duplicate email records while keeping the row with the smallest ID.

**Time Complexity:** O(n log n)

**Space Complexity:** O(1)

## Problem 2

**Problem Name:** Design Parking System

**Platform:** LeetCode

**Language:** Java

**Difficulty:** Easy

**Approach:** Use object-oriented programming to keep track of the available parking slots for each car type and update the count whenever a car is parked.

**Time Complexity:** O(1)

**Space Complexity:** O(1)

## Problem 3

**Problem Name:** Remove Element

**Platform:** LeetCode

**Language:** Java

**Difficulty:** Easy

**Approach:** Use the Two Pointer technique to overwrite elements that are not equal to the given value and return the new length of the modified array.

**Time Complexity:** O(n)

**Space Complexity:** O(1)
